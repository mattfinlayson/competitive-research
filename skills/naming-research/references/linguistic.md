# Linguistic & Cultural Analysis Reference

Pronunciation, spelling, and international considerations for naming.

## Overview

A name must be:
1. **Sayable**: People can pronounce it correctly on first read
2. **Typeable**: People can spell it after hearing it
3. **Universal**: It doesn't mean something unintended abroad

## Pronunciation Clarity

### The Phone Test

Read the name aloud. Ask:
1. Is there only one obvious way to pronounce it?
2. Can someone pronounce it correctly after seeing it once?
3. Does pronunciation vary by accent?

**Examples**:

| Name | Pronunciation | Clarity |
|------|---------------|---------|
| Attest | uh-TEST | Clear |
| Vanta | VAN-ta | Clear |
| Osano | oh-SAH-noh | Requires guidance |
| Salesforce | salz-FORSS | Clear |
| GitHub | GIT-hub | Clear |

**Problem patterns**:
- Silent letters: Knite, Pneuma
- Multiple vowel sounds: Aerie, Ioi
- Ambiguous consonants: Xeer, Zoi

### Phonetic Spelling

If pronunciation isn't obvious, add guidance:

```
Pronunciation: "at-TEST" (rhymes with "request")
```

Include this in brand guidelines.

## Spelling Clarity

### The Dictation Test

Say the name aloud. Ask someone to spell it:

1. Can they spell it after hearing it once?
2. Are there common misspellings?
3. Are there phonetic alternatives?

**Problem patterns**:

| Name | Issue | Common Misspellings |
|------|-------|---------------------|
| Attest | Clear | None |
| Xero | Hard to spell | Zero, Xiro |
| Flickr | Easy to confuse | Flicker |
| Tumblr | Easy to confuse | Tumbler |

### Tips for Spelling Clarity

1. **Use common letter patterns**: "Attest" > "At3st"
2. **Avoid silent letters**: "Color" > "Colour" (in US)
3. **Single pronunciation per letter**: Avoid "ph" for "f"
4. **Standard spelling**: Prefer dictionary words

## International Considerations

### The Translation Test

Translate the name to major market languages:

| Language | Key Questions |
|----------|--------------|
| French | Does it mean anything vulgar or negative? |
| German | Does it combine with other words awkwardly? |
| Spanish | Does it mean anything inappropriate? |
| Japanese | Can it be written phonetically? |
| Chinese | Can it be written with positive characters? |

### Common International Issues

**False Friends** (words that look similar but mean different things):

| Name | English | German |
|------|---------|--------|
| Gift | Present | Poison |
| Bra | Underwear | Bridge |
| Butt | Rear | Barrel |

**Negative Meanings**:

| Name | English | Issue |
|------|---------|-------|
| Nova | New star | - |
| Nova | - | "Doesn't go" in some Slavic languages |

**Cultural Considerations**:

| Name | Consideration |
|------|---------------|
| Numbers | 4 is unlucky in East Asia |
| Colors | White = death in some cultures |
| Animals | Dog = disloyal in some cultures |

### Mitigation Strategies

1. **Phonetic transliteration**: Write the name in each script
2. **Pronunciation guide**: Provide for non-native speakers
3. **Focus groups**: Test with native speakers
4. **Avoid obvious false friends**: Screen against major languages

## Scoring Rubric

| Score | Criteria |
|-------|----------|
| 5 | Perfect: clear pronunciation, spelling, no international issues |
| 4 | Minor issues: pronunciation guidance needed, no international problems |
| 3 | Moderate issues: spelling not obvious OR one international concern |
| 2 | Significant issues: multiple problems with pronunciation/spelling/international |
| 1 | Major issues: unusable due to pronunciation, spelling, or meaning problems |

## Length Considerations

### Ideal Length

- **Syllables**: 2-3 syllables ideal
- **Characters**: Under 10 characters preferred
- **Words**: Single word ideal (or two short words)

| Length | Example | Assessment |
|--------|---------|------------|
| 1-4 chars | Vow, Att | Too short, availability issues |
| 5-7 chars | Attest, Certify | Ideal range |
| 8-10 chars | Compliance | Acceptable |
| 11+ chars | SubProcessor | Too long |

### Syllable Count

| Syllables | Example | Feel |
|-----------|---------|------|
| 1 | Vow | Punchy, memorable |
| 2 | Attest | Balanced |
| 3 | Certify | Professional |
| 4 | Compliance | Formal, heavy |

## Documentation

For each candidate, document:

1. **Pronunciation**: How is it pronounced?
2. **Spelling**: Common misspellings?
3. **International**: Translation issues?
4. **Length**: Syllable and character count
