# Trademark & Domain Clearance Reference

Methodology for trademark and domain availability analysis.

## Overview

Trademark and domain availability is often the deciding factor in naming. This reference provides a framework for systematic analysis.

## The Three-Step Process

### Step 1: Preliminary Search

Before deep analysis, run quick searches:

1. **Google search**: Search for the name + "software" or "company"
2. **LinkedIn search**: Check for companies with similar names
3. **GitHub search**: Check for organizations with similar names

Purpose: Identify obvious conflicts early.

### Step 2: Trademark Search

#### USPTO TESS (Trademark Electronic Search System)

1. Go to: https://tmsearch.uspto.gov/
2. Select "Basic Word Mark" search
3. Enter the name (without TLD)
4. Filter for software-related classes (Class 9, 42, 35)

**What to look for**:
- Exact matches (likely blockers)
- Phonetically similar names
- Visually similar names
- Same class registration

#### Common-Law Search

Trademarks can exist without registration:

1. Search state business registries
2. Search business directories (Yelp, Yellow Pages)
3. Search industry publications

**Red flags**:
- Company with same name in same industry
- Established brand with similar name
- Pending trademark applications

### Step 3: Domain Availability

Check these TLDs in order of priority:

| TLD | Priority | Notes |
|-----|----------|-------|
| .com | 1st | Most valuable, usually first choice |
| .io | 2nd | Tech/SaaS standard |
| .ly | 3rd | Trendy, short names |
| .co | 4th | Alternative to .com |
| .app | 5th | Modern, app-focused |

**Tools**:
- Namecheap
- Hover
- GoDaddy
- Instantdomainsearch

**Social Handle Availability**

| Platform | Importance | Check |
|----------|------------|-------|
| Twitter/X | High | @name |
| LinkedIn | High | linkedin.com/name |
| GitHub | High | github.com/name |
| Instagram | Medium | @name |

**Tools**:
- Namecheckr
- Knowem
- SocialSearch

## Scoring Rubric

| Score | Criteria |
|-------|----------|
| 5 | .com available, all social clear |
| 4 | .com taken but .io/.ly available, social mostly clear |
| 3 | .com taken, other TLDs available, some social conflicts |
| 2 | Limited TLD availability, significant social conflicts |
| 1 | .com + .io both taken, major social conflicts |

## Common Issues

### Dictionary Words

Many short names are dictionary words:
- "Attest" - may conflict with attestation services
- "Vouch" - may conflict with voucher services
- "Prove" - may conflict with proof/verification services

**Mitigation**:
- Add suffix (.ly, .app)
- Add context (company name vs. product name)
- Check class-specific trademarks

### International Conflicts

A name clear in the US may conflict abroad:
- Check EUIPO (European Union)
- Check UKIPO (United Kingdom)
- Check local registries for key markets

### Typo Squatting

If your name is "Attest", check:
- attestt.com
- atesty.com
- attest.ly
- at-test.com

## Decision Framework

```
IF .com available AND social clear:
  → Strong candidate
ELIF .io/.ly available AND social clear:
  → Good candidate (with branding adjustment)
ELIF other TLDs available but social conflicts:
  → Evaluate conflict severity
  → Consider rebrand if major conflicts
ELSE:
  → Strongly consider alternatives
```

## Documentation

For each candidate, document:

1. **Trademark status**: Clear, potential conflict, clear conflict
2. **Domain availability**: .com, .io, .ly, .co, .app
3. **Social availability**: Twitter, LinkedIn, GitHub, Instagram
4. **Other registrations**: EU, UK, key markets
